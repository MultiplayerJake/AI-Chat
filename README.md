# AI-Chat
AI chat with the use of Ollama. I used Qwen3.5 and llama3.1, but you can use any you install from Ollama.
You need to install ollama on https://ollama.com/, then any model from https://ollama.com/search, but I recommend Qwen3.5 or llama3.1.
Then, in Windows PowerShell, put this: ollama run "the model you use." or look here https://docs.ollama.com/quickstart.
You need to use it with the Live Server extension in VS Code.
After you put your model name in the model name input, and if you want, a system prompt in the system prompt input, you are ready to go.
